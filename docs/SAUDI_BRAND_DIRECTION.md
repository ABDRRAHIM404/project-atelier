# Saudi Brand Direction

The customer-facing UI uses a Saudi/Gulf luxury direction while preserving the approved “بيتي بذوقي” logo.

## Palette

- Rich teal: `#0F766E` — primary actions and selected states.
- Sand gold: `#D2A94A` — decorative highlights only.
- Warm clay: `#A86745` — furniture and architectural accents.
- Warm cream: `#FFF8ED` — primary background.
- Deep brown: `#3F3028` — headings and body text.

## UX rules

- White text on rich teal is used for primary buttons.
- Sand gold is not used for essential text because it lacks sufficient contrast on cream.
- Deep brown remains the main reading color.
- Decorative geometry should be restrained and inspired by contemporary Saudi/Najdi forms, not Moroccan riad, zellige, or lantern motifs.
